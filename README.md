# BanZZak
> 동작 인식을 통해 알람을 해제하고, 포인트를 획득하는 IoT 연동 앱

## 2020 Super Challenge Hackathon Team - BanZZak
> 김보성, 김영규, 김현탁, 김희연, 이용광, 최승우

## 개발 환경
> Android Studio 3.5.3

## 언어 및 사용 라이브러리
> Java, Kotlin
> Tensorflow lite, Glide, Room, ViewModel, BluetoothKit
